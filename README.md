async-examples
==============

Async.js Quick Examples

## Usage

Install [CoffeeScript](http://coffeescript.org) if not installed.

   ```bash
   $ sudo npm install -g coffee-script
   ```

Then,

   ```bash
   $ git clone git://github.com/philipjkim/async-examples.git
   $ cd async-examples
   $ npm install
   $ cake run
   ```
